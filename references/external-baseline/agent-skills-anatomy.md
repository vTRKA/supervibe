# Agent Skills External Baseline Anatomy

The reusable baseline for agent/skill audits is advisory but enforced through local validators where the repository has adopted the rule.

Required anatomy signals:

- Source-of-truth preflight.
- When-not-to-use boundary.
- Decision tree.
- Procedure.
- Common rationalizations.
- Red flags.
- Checklist.
- Failure modes.
- Output contract.
- Verification commands.
- Related skills, agents, rules, or reference packs.
