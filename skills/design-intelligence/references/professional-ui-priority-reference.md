# Professional UI Priority Reference

Review order for design intelligence:

1. Accessibility and contrast
2. Touch, keyboard, focus, and interaction states
3. Performance and render cost
4. Style selection and brand fit
5. Layout and responsive behavior
6. Typography and color hierarchy
7. Motion and transitions
8. Forms, feedback, empty, loading, and error states
9. Navigation and wayfinding
10. Charts and data comprehension

Each finding needs owner agent, severity, platform scope, and anti-pattern.
