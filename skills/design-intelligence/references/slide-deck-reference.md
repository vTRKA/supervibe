# Slide Deck Reference

Slide guidance supports presentation strategy and deck implementation.

Rules:
- Presentation-director owns audience, narrative arc, emotion arc, copy strategy, and slide count.
- Presentation-deck-builder owns layout, visual hierarchy, chart selection, and token-compliant implementation.
- Chart slides require fallback text and accessibility notes.
- Avoid generic deck advice when slide strategy/layout/copy rows provide a better match.
