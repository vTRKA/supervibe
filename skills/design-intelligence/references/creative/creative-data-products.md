# Creative Reference Pack: Data Products

Reference role: interaction benchmark
Quality tier: tier-2 local curated pack
Best for: fast path, medium path, full creative path

Creative moves:
- Make density feel intentional through stable grid rhythm, semantic color,
  progressive disclosure, and comparison-first layout.
- Use chart shells, table controls, filters, drilldowns, and annotations as
  brand-bearing components.
- Design loading, empty, partial, error, and export states as first-class
  surfaces.

Borrow:
- Dashboards, analytics products, finance tools, admin consoles, observability,
  research, and operations workflows.
- Dense but calm scanning patterns.
- Data ink hierarchy, not decorative chart color.

Avoid:
- Hero-scale typography inside compact operational panes.
- Color-only meaning, hidden legends, chart junk, and unstable table columns.
- Over-polishing marketing surfaces while core data workflows stay generic.

Differentiation pressure:
- Use when the product must feel expert without becoming visually loud.
- Combine with editorial or luxury packs only when density remains usable.

Do not use as style authority:
- This pack provides pattern ingredients, not a product to copy.
