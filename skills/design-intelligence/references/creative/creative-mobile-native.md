# Creative Reference Pack: Mobile Native Tactility

Reference role: interaction benchmark
Quality tier: tier-2 local curated pack
Best for: fast path, medium path, full creative path

Creative moves:
- Use thumb-zone composition, tactile feedback, native sheets, safe areas, and
  stateful transitions as the design language.
- Make empty, loading, offline, permission, and recovery states feel designed.
- Use subtle motion and haptics as confirmation, not spectacle.

Borrow:
- Mobile onboarding, daily-use dashboards, creator tools, finance companions,
  and field workflows.
- Native affordances that reduce cognitive load.
- Bottom sheets, segmented controls, swipe actions, and progressive disclosure
  when they match the task.

Avoid:
- Porting desktop layout density directly to mobile.
- Decorative motion that fights scroll performance or reduced motion.
- Hiding destructive or irreversible actions behind gestures alone.

Differentiation pressure:
- Use when competitors ship web views with mobile chrome.
- Pair with regulated trust for high-stakes mobile flows.

Do not use as style authority:
- This pack provides pattern ingredients, not a product to copy.
