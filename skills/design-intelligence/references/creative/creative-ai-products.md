# Creative Reference Pack: AI Products

Reference role: interaction benchmark
Quality tier: tier-2 local curated pack
Best for: medium path, full creative path

Creative moves:
- Design trust around uncertainty: source evidence, confidence, editable
  outputs, rollback, preview, and critique states.
- Make agent activity visible without turning the UI into noisy telemetry.
- Use generative outputs as drafts with review, compare, accept, and reject
  affordances.

Borrow:
- Copilot workflows, agent consoles, prompt-to-artifact tools, model evaluation
  surfaces, and AI-assisted creative products.
- Clear boundaries between suggestion, action, and committed change.
- Side-by-side comparison and provenance moments.

Avoid:
- Chat-only shells for non-chat work.
- Magical claims, hidden mutation, unreviewed automation, and vague confidence.
- Reusing generic sparkle or gradient language as the identity.

Differentiation pressure:
- Use when AI competitors converge on the same chat panel and purple glow.
- Pair with devtools, data products, or regulated trust based on risk.

Do not use as style authority:
- This pack provides pattern ingredients, not a product to copy.
