# Creative Reference Pack: Experimental Web

Reference role: creative benchmark
Quality tier: tier-2 local curated pack
Best for: full creative path

Creative moves:
- Use scroll-state, spatial layering, procedural texture, 3D depth, and
  non-linear reveals to create a memorable first impression.
- Make interaction the identity: cursor behavior, staged disclosure, depth,
  canvas motion, or responsive soundless choreography.
- Let the visual system express one strong metaphor tied to the product.

Borrow:
- Launch pages, product reveals, interactive explainers, creative tools,
  culture products, and high-energy campaigns.
- Immersive background systems when they reveal product state or value.
- Motion as structure, not decoration.

Avoid:
- Blocking reading, conversion, keyboard access, reduced motion, or mobile
  performance.
- Using experimental mechanics on routine admin or support workflows.
- Treating novelty as quality without product fit.

Differentiation pressure:
- Use when the category has identical cards, centered heroes, and predictable
  scroll sections.
- Pair with accessibility and performance checks before approval.

Do not use as style authority:
- This pack provides pattern ingredients, not a product to copy.
